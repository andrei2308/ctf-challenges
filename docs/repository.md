# Repository Overview

This repository collects personal CTF writeups and challenge notes across multiple platforms and competitions. The emphasis is on methodology: not just the final exploit or recovered flag, but the decisions, pivots, tooling, and constraints that made the solve work.

## Coverage

- Binary exploitation and memory corruption
- Reverse engineering on Linux, Windows, and Android targets
- Web exploitation and browser-side attack chains
- Cryptography and applied cryptanalysis
- Forensics, OSINT, and threat hunting

## What To Expect

- Step-by-step exploitation or analysis paths
- Practical notes on tools and environment setup
- Intermediate observations, not just final payloads
- Reusable patterns that transfer to similar challenges

These writeups are for educational use. The repository only documents retired challenges, explicitly shareable content, or competition material where publication is allowed.
